📊 Netflix Data Analysis Project

🔍 Objective
Analyze Netflix dataset to derive insights about content distribution, genres, countries, and growth trends.

🛠 Tools Used
Python
Pandas
Matplotlib
Jupyter Notebook

📈 Key Insights
Netflix has more Movies than TV Shows.
United States produces the highest content.
Content additions increased significantly after 2016.
Drama and International Movies dominate the platform.

📁 Dataset
Netflix Titles Dataset (Kaggle)
